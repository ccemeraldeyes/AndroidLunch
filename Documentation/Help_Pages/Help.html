Add a category, list item, or tag

1.       Press the device menu button and select Add.

2.       See Add help page.

View a category list

1.       Category lists are displayed by default. If tag names are displayed in the horizontal tabs, press the device menu button and select View by Categories.

2.       The tab of the current list being displayed is highlighted.

3.       A different category list can be viewed by pressing the tab of the desired list.

Add Page

To reach this page

1.       Press the device Menu button

2.       Select Add

 

Add a main category

1.       Select Add a Category.

2.       Enter the category information into the fields provided. Fields marked with a * are required.

3.       Press color to bring up a color chooser, then select a color that will be associated with this category. This Choosing distinct colors for each category is recommended.

4.       Press the Add button when finished.

Add a tag

1.       Select Add a Tag.

2.       Enter the tag name into the field provided.

3.       Press the Add button when finished.

Add a list item

1.       Select Add a List Item.

2.       Select a category for the item.

3.       Enter the item information into the fields provided. Fields marked with a * are required.

4.       Press the Add button when finished.
